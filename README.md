# Cluster

**A wall of remote desktops.** Cluster shows up to 36 live RDP connections at once in a clean grid —
a security-camera wall for your PCs, servers, and VPSes. Every viewport is live but view-only; expand
one for full control, go true-fullscreen, or yank a single screen out into its own floating window.

Free to use. Download it from **[kvrnl.io/products/cluster](https://kvrnl.io/products/cluster/)**.

## Features

- **2×2 up to 6×6 grid** — switch layouts instantly in Settings.
- **Live view-only wall** — see every machine at a glance; click to take control.
- **True fullscreen** on any viewport, and **pop-out** to a free-floating desktop window.
- **This-PC panel** — shows your own machine's address, name, and username so you always know what to connect to.
- **Drop alerts** — a toast pops the moment a connection drops.
- **System tray**, saved connections (passwords encrypted with Windows DPAPI), and silent auto-updates.

## Requirements

- **Windows 10/11 (x64).** Self-contained build — no separate .NET install required.
- Uses the built-in Windows Remote Desktop client control; target PCs need Remote Desktop enabled.
